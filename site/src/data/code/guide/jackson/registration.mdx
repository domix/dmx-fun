```java
// Manual registration
ObjectMapper mapper = new ObjectMapper()
    .registerModule(new DmxFunModule());

// Auto-discovery (recommended) — works because DmxFunModule is registered
// as a com.fasterxml.jackson.databind.Module service provider via META-INF
ObjectMapper mapper = new ObjectMapper().findAndRegisterModules();

// Spring Boot with fun-spring-boot — no registration needed.
// DmxFunJacksonAutoConfiguration registers DmxFunModule as a bean automatically
// when both jackson-databind and fun-jackson are on the classpath.

// Spring Boot without fun-spring-boot — declare the bean manually;
// Spring's JacksonAutoConfiguration applies it to the ObjectMapper.
@Bean
public DmxFunModule dmxFunModule() {
    return new DmxFunModule();
}
```
